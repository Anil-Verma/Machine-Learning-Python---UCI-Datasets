# Machine-Learning-Python---UCI-Datasets
This repository contains the machine learning model of the open datasets in UCI. Here focus is on following the best practices of data science.

`Data Source: https://archive.ics.uci.edu/ml/datasets.php`

## The repository is created after going the following process of all datasets:
1. Data Gathering
2. Data Cleaning
3. Feature Engineering
4. Data Visualization
5. Data Modelling
6. Model Comparison
7. Prediction (if Applicable)

### The Folders are arranged in the form of models that are made. These are:
- Regression (and its types)
- Classification (and its types)
- Clustering (and its types)
